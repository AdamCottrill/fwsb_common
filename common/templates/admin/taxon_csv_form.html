Add a New Taxon Entry

This page can be used to insert new taxon objects to the taxon table. The file must be a csv, with these (lower case) headings:

  • taxon
  • itiscode
  • taxon_name
  • taxon_label
  • taxonomic_rank
  • vertinvert
  • parent_tsn

csv files for this upload can be created from R-script which compiles the records from the ITIS database. Each record represents an entry in the taxonomic tree and they must be ordered to reflect the taxonomic hierarchy - with the most detailed level last. Taxomnic entries that already exist will be ignored, but new ones will be created.
